# Drone Scope — Website Premium

Website oficial pentru Drone Scope — filmări și fotografii aeriene premium din Constanța.

## Paletă
- Fond: #0B0B0E
- Accent principal: #14B8A6 (teal/cyan cinematic)

## Tehnologii
- Next.js 15 + TypeScript
- Tailwind + design system custom
- Framer Motion (animații)
- React Hook Form + Zod (formular)
- Sonner (toast-uri)

## Cum rulezi local

```bash
npm install
npm run dev
```
